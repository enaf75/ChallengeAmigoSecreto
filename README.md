<h1 align="center"> Challenge: Amigo Secreto </h1>


## :interrobang: Descripción del proyecto :interrobang:
El Juego del Amigo Secreto es un proyecto desarrollado de la mano con ALURA LATAM para fortalecer nuestra lógica de programación mediante la creación del "Juego del Amigo Secreto", 
el cual permite ingresar los nombres de los amigos y seleccionar un ganador mediante un número aleatorio.
